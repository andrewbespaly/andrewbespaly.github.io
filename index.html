Subdivision


Subdivision is an algorithm that takes an object made of triangles and creates four new smaller triangles for every original triangle. Every iteration makes the object become either more smooth or more detailed depending on the users goal in mind. This was my implemenation of subdivision used to create a realistic mountain rendering.
